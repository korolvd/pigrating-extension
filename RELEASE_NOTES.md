PigRating — соцрейтинг свиноколеса для [pointauc.com](https://pointauc.com/).
Перед прокрутом заливает баллы из Google-таблицы в лоты через официальный API.

**Возможности:**
- авто «+ к лоту» для одиночного вкладчика; спорные / групповые / без лота — выбор лота прямо в окне;
- подпись «от [метка] ник:сумма» на лоте + защита от задвоения («⚠ уже залито»);
- **откат соцрейтинга**: всё, выборочно (галочки) или конкретного ника (инлайн `↩`);
- плюс/минус, режим «как донат», фоновая заливка одним батчем.

**Установка:** скачайте `pigrating.zip` → распакуйте → `chrome://extensions` → Режим разработчика → «Загрузить распакованное» → выберите папку.

Подробнее — в [README](https://github.com/korolvd/pigrating-extension#readme).
