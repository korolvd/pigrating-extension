# PigPoints v0.6.0

Большое обновление с 0.3.0: расширение переосмыслено как **PigPoints** — значки Twitch, балы канала и Google-таблица превращаются в ставки в pointauc. (Версии 0.4.x–0.5.x отдельно не выкатывались — здесь всё сразу.)

---

## 🏅 Ставка за лот (значки + PigPoints) — флагман

Зритель за **1 балл канала** активирует «Предложить лот», вписывает название — в pointauc уходит ставка **= база + цены его значков Twitch + его PigPoints из таблицы**, в очередь подтверждения (`insertStrategy: none`).

- **Пул значков с ценами:** саб 1/2/3, VIP, модератор, фолловер, топ-1/2/3 даритель, топ-1/2/3 клипер, топ-1/2/3 по битам, sub-gifter по тирам, биты по тирам, founder. Саб-тир/VIP/мод/фолловер — свежим запросом Helix в момент активации; остальные — из значков сообщения награды.
- **PigPoints в ставке:** плюс — прибавляется всегда; минус — только в свой новый лот или где зритель единственный вкладчик (в общий лот / поддув — не идёт). Тумблер **«не учитывать отрицательные PigPoints в общих лотах»** (вкл) можно выключить. «Свой/чужой» определяется по текущей доске pointauc.
- **Анти-повтор:** значки и PigPoints за раунд считаются один раз; **журнал ставок** показывает разбивку (база · значки · PigPoints, 🟢 свой / 🟡 поддув). Учёт раунда сбрасывается выключением/включением тумблера.

## 🎁 Покупка PigPoints за балы канала

Зрители тратят балы канала и меняют свои PigPoints — пишется в Google-таблицу (ту же, что питает ставку за лот). Модуль **«PigPoints · таблица»**.

- **Запись через Apps Script** (без Google Cloud): «📋 Скопировать скрипт» подставит настройки и сгенерит секрет; «❔ Как установить?» со скриншотами; индикатор 🟢/🔴 ловит устаревший скрипт.
- **Twitch в один клик** (приложение встроено). **Награды** задаёшь таблицей (название · цена в балах · PigPoints · кому).
- **Авто-начисление** известных ников; подозрительные/похожие — на ручное подтверждение; **подсказка похожего ника** (опечатка → ближайший ник из таблицы); **кнопка «очистить журнал»**.

## 🔧 Прочее в 0.6.0

- **Ручной залив (Предпросмотр / Применить / Откатить) убран из интерфейса** — его заменяет авто-ставка за лот. Логика оставлена в коде как `@deprecated` (на удаление).
- **Ребрендинг:** соцрейтинг/рейтинг → **PigPoints**; метка `[СОЦРЕЙТИНГ]` → `[PP]`; модуль «PigRating» → «PigPoints · таблица»; поле Twitch Client ID убрано.
- Редизайн popup (блок «Источники ставок»), сгруппированные настройки, новый логотип.

## Установка / обновление
1. Скачай **`pigrating.zip`** → распакуй → `chrome://extensions` → Режим разработчика → «Загрузить распакованное» (при обновлении — просто ↻).
2. Вкладку аукциона держи открытой (API шлёт команды в неё).
3. **Переподключи Twitch** («Подключить Twitch») — нужны скоупы `channel:manage:redemptions`, `user:read:chat`, `channel:read:subscriptions`, `channel:read:vips`, `moderation:read`, `moderator:read:followers`.
4. Для записи в таблицу — поставь Apps Script («❔ Как установить?»). Награды Twitch требуют статуса **Affiliate/Partner**.
