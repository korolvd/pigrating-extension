# PigPoints v1.0.0

Первый стабильный релиз. Если обновляешься с опубликованной **0.5.1** — здесь всё из 0.6.0 (соцрейтинг встроен прямо в ставку, ребрендинг в **PigPoints**) плюс доводка 1.0: единая терминология, обобщение под любые лоты, каскад настроек с подсказками, полное руководство и чистка кода.

---

## 🏅 Ставки в pointauc — флагман

Зритель за **1 балл канала** активирует награду «Предложить лот», вписывает название (фильм, игру — что угодно) — в pointauc уходит ставка **= база + цены его значков чата + его PigPoints из таблицы**, в очередь подтверждения (`insertStrategy: none`).

- **Пул значков чата с ценами:** саб 1/2/3, VIP, модератор, фолловер, артист, кондуктор хайп-трейна, топ-1/2/3 даритель, топ-1/2/3 клипер, топ-1/2/3 по битам, sub-gifter по тирам, биты по тирам, founder. Саб-тир/VIP/мод/фолловер — свежим запросом Helix в момент активации; остальные — из значков сообщения награды.
- **PigPoints в ставке:** плюс — прибавляется всегда; минус — только в свой новый лот или где зритель единственный участник (галка «Отрицательные PigPoints — только в свой лот», по умолчанию вкл).
- **Анти-повтор + журнал ставок** с разбивкой (база · значки · PigPoints, 🟢 свой / 🟡 поддув).

## 🐷 Покупка PigPoints за баллы канала

Зрители тратят баллы канала и получают PigPoints — пишутся в Google-таблицу (ту же, что питает ставки). Модуль **«PigPoints»**.

- **Запись через Apps Script** (без Google Cloud): «📋 Скопировать скрипт» подставит настройки и сгенерит секрет; «❔ Как установить?» со скриншотами; индикатор ловит устаревший скрипт.
- **Награды** задаёшь таблицей (название · цена в баллах · PigPoints · кому); авто-начисление известных ников, ручное подтверждение подозрительных, подсказка похожего ника.
- 💡 PigPoints можно вести и **вручную** в таблице — например, учитывать поддержку с других платформ (Boosty, Patreon): вписал зрителю PigPoints — его ставки весомее.

## ✨ Что нового в 1.0.0

- **Единая терминология:** «значки чата» (термин Twitch); «лот» вместо «фильм» — фича для любой тематики аукциона.
- **Понятные названия** модулей и разделов; журналы «Журнал ставок pointauc» / «Журнал покупок PigPoints»; очередь «Покупки на подтверждение».
- **Каскад настройки с подсказками:** чтение → (PigPoints в ставке + запись) → награды. Неготовый блок приглушён и подсказывает, что подключить; модуль «PigPoints» прячется, пока не настроен весь конвейер.
- **Точный хелсчек скрипта:** смена имени листа сразу перепроверяется; расхождение с задеплоенным скриптом → «⚠ скрипт устарел — пере-скопируй и задеплой».
- **Логотипы** (свиноколесо / колесо pointauc), индикатор ⏻ у мастер-тумблеров, «✓ Автосохранено».
- **Полное руководство** по расширению — «?» в шапке (обзор, подключения, обе механики, установка, траблшутинг).
- Под капотом: удалён старый код ручного залива/отката — ставки идут только через `POST /bids`.

## Установка / обновление (ручная)
1. Скачай **`pigrating.zip`** → распакуй → `chrome://extensions` → Режим разработчика → «Загрузить распакованное» (при обновлении — просто ↻).
2. Вкладку аукциона держи открытой (API шлёт команды в неё).
3. **Переподключи Twitch** («Подключить Twitch») — нужны скоупы `channel:manage:redemptions`, `user:read:chat`, `channel:read:subscriptions`, `channel:read:vips`, `moderation:read`, `moderator:read:followers`.
4. Для записи в таблицу — поставь Apps Script («❔ Как установить?»). Награды Twitch требуют статуса **Affiliate/Partner**.

Полное описание — в [README](https://github.com/korolvd/pigrating-extension#readme) · детальная история по версиям — в [CHANGELOG](https://github.com/korolvd/pigrating-extension/blob/master/CHANGELOG.md).
