# PigRating v0.5.1

Большое обновление с 0.3.0 — **две новые механики автоматического добавления ставок в pointauc из Twitch**, плюс доводка после боевых тестов. (Версии 0.4.0–0.5.0 отдельно не выкатывались — здесь всё сразу.)

---

## 🎁 Покупка рейтинга за балы канала Twitch

Зрители тратят **балы канала** и меняют соцрейтинг (себе или другому) — начисления пишутся в **ту же Google-таблицу**, которую расширение потом применяет к лотам.

- **Запись в таблицу через Apps Script** (без Google Cloud): кнопка «📋 Скопировать скрипт» подставит твои настройки и сгенерит секрет; индикатор 🟢/🔴; пошаговая инструкция «❔ Как установить?» со скриншотами прямо в расширении.
- **Twitch в один клик** — «Подключить Twitch» (приложение уже встроено, регистрировать ничего не надо).
- **Награды** задаёшь таблицей (название · цена в балах · рейтинг · кому); расширение само создаёт/обновляет их на канале. Тумблер **«Награды за балы канала»** наверху.
- **Кому:** себе или по нику из поля награды (подарок/диверсия); минус-награды = понижение.
- **Авто-начисление** (по умолчанию вкл): известные ники начисляются сразу; **подозрительные/новые** — на ручное подтверждение (✓ начислить / ✕ вернуть баллы). Тумблер **«auto»** в очереди.
- **Подсказка похожего ника** (как «похожий лот» в pointauc): опечатался зритель (`xsalreen` вместо `xalreenstream`) — плагин предложит ближайший ник из таблицы, начислишь одним кликом.
- `@` и регистр в никах не важны; новые ники пишутся после последнего ника; ошибки возврата видны в логе; журнал последних начислений `@покупатель → ник`.

## 🏅 Ставка за значки на фильм

Отдельная механика (без таблицы): зритель за **1 балл** активирует награду **«Предложить фильм»**, вводит название, и в pointauc уходит ставка **= база + сумма цен его значков Twitch**. Значки = социальный вес заявки.

- **Пул значков с ценами** в настройках: саб по тирам 1/2/3, VIP, модератор, фолловер, **топ-1/2/3 даритель**, **топ-1/2/3 клипер**, **топ-1/2/3 по битам**, sub-gifter по тирам, биты по тирам, founder. Название награды и базу задаёшь сам.
- **Свежие статусы:** саб-тир/VIP/мод/фолловер запрашиваются в момент активации (ловит подписки «ради участия» по ходу аука); топ-даритель/клипер/биты и прочие — из значков сообщения.
- **Ставка → очередь подтверждения pointauc** (как донат): принимаешь сам, pointauc предлагает похожий лот. Галка **«как донат»** — слать с конвертацией аука или 1:1.
- **Анти-повтор:** считаются только новые значки зрителя; повтор без новых → возврат балла; кнопка «Новый раунд». **Журнал ставок** с разбивкой.

## 🛠 Прочее
- **Откат рейтинга** прямо с доски (всё/выборочно/по нику), стейтлес по меткам.
- Хелсчек веб-аппа ловит **устаревший Apps Script** (столбцы/лист разошлись с настройками → «⚠ пере-скопируй и задеплой»).
- Покупки можно писать в **отдельный столбец** (галка), чтобы не трогать формульный столбец рейтинга.
- Удалённая в плагине награда убирается и с Twitch; зачистка «осиротевших», переиспользование одноимённых (без дубликатов).
- Новый логотип (радужное колесо + свинка), страница установки со скриншотами.

## Установка / обновление
1. Скачай **`pigrating.zip`** → распакуй → `chrome://extensions` → Режим разработчика → «Загрузить распакованное» (при обновлении — просто ↻).
2. Вкладку аукциона держи открытой (API шлёт команды в неё).
3. **Для покупки рейтинга / ставок за значки — переподключи Twitch** («Подключить Twitch»): нужны новые права `user:read:chat`, `channel:read:subscriptions`, `channel:read:vips`, `moderation:read`, `moderator:read:followers`, `channel:manage:redemptions`.
4. Для записи в таблицу — поставь Apps Script («❔ Как установить?»). Награды Twitch требуют статуса **Affiliate/Partner**.

Полное описание — в [README](https://github.com/korolvd/pigrating-extension#readme) · детальная история по версиям — в [CHANGELOG](https://github.com/korolvd/pigrating-extension/blob/master/CHANGELOG.md).
